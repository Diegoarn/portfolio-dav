Aplicación web Inmobiliaria

Trabajo de fin de curso del bootcamp de JavaScript que consistía en crear un web para una empresa inmobiliaria.
Aplicación web dinámica, con gestión de catálogo y usuarios. Aplicación cliente-servidor diseñada en tres capas y desarrollada en un nivel.
Capa de presentación, HTML, CSS y JavaScript.
Capa de aplicación con JavaScript.
Capa de Datos, persistencia de datos con archivos JSON.
