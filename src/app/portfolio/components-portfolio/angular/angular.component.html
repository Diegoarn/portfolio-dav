Aplicación web Angular

Aplicación web dinámica, desarrollada con el framework Angular 14 y lenguaje TypeScript, siguiendo una estructura orientada a módulos y servicios. Desde los servicios se accede a la Api de Rest para el intercambio de datos, definiendo JSON cómo estándar de comunicación, para la creación de galerías dinámicas.
